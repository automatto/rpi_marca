# RpiMarca

Leitura da RPI de Marcas do INPI em formato XML.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rpi_marca'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rpi_marca
